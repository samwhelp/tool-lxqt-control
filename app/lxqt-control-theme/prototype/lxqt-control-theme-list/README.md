

## Build Essential

``` sh
$ sudo apt-get install build-essential
```

``` sh
$ sudo apt-get install liblxqt0-dev
```


## How to build

Create Makefile

``` sh
$ qmake
```

Then

``` sh
$ make
```


## How to test

``` sh
$ ./lxqt-control-theme-list
```
